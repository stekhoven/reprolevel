# Examples

This directory is available for optional community examples and case studies showing how the ReproLevel framework has been applied in practice.

Potential uses include:

- discipline-specific worked examples;
- example self-assessments of projects or labs;
- teaching materials built around the levels;
- short case studies showing how a project moved from one level to another.

If examples are added here, they should avoid sharing sensitive data or confidential project details without permission.
